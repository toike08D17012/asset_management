# asset_management - Technology Stack

## 言語

- TypeScript 5.7+
- Node.js 22+

## フレームワーク

**フロントエンド**:
- Next.js 15 (React 19, App Router)
- Tailwind CSS 3.4
- Recharts 2.15 (グラフ描画)

**バックエンド**:
- Next.js API Routes (Server-side)
- Drizzle ORM 0.38

**データベース**:
- SQLite (better-sqlite3) - 個人利用のため軽量DB採用

**セキュリティ**:
- AES-256-GCM (Node.js crypto)
- PBKDF2 (マスターキー導出)

## ツール

- Vitest (テスト)
- ESLint (リント)
- Tailwind CSS (スタイリング)
- Drizzle Kit (DB マイグレーション)
